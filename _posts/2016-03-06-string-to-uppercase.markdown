This example shows how to uppercase every letter in a string.

{% highlight elixir %}
"HELLO WORLD!" = String.upcase("Hello World!")
{% endhighlight %}

Documentation: [String.upcase/1](https://hexdocs.pm/elixir/String.html#upcase/1)
