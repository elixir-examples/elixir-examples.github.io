This example shows how to lowercase every letter in a string.

{% highlight elixir %}
"hello world!" = String.downcase("Hello World!")
{% endhighlight %}

Documentation: [String.downcase/1](http://elixir-lang.org/docs/stable/elixir/String.html#downcase/1)
